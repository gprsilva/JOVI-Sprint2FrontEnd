Qual é a derivada de f(x) = x³ + 2x²?

Aplicando a regra da potência:
f'(x) = 3x² + 4x
Detalhando cada etapa:
1Derivada de x³ → baixa o expoente: 3x² 2Derivada de 2x² → 2·2x = 4x

E se eu precisar da segunda derivada?

JOVI EDU IA
Derivamos f'(x) mais uma vez:
f''(x) = 6x + 4
A segunda derivada indica a concavidade da curva. Quando f''(x) > 0 a parábola é côncava para cima, quando < 0 é côncava para baixo.
